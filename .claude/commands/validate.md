# Команда: /validate

> Запускает Валидатора для финальной проверки.

---

## Синтаксис

```bash
/validate
```

---

## Описание

Команда `/validate` выполняет:
- Проверку всех предыдущих ворот
- Верификацию всех артефактов
- Обновление RTM
- Формирование итогового отчёта

---

## Агент

**Валидатор** (`.claude/agents/validator.md`)

---

## Предусловия

| Ворота | Требование |
|--------|------------|
| `QA_PASSED` | Тесты пройдены, покрытие ≥75% |

---

## Выходные артефакты

| Артефакт | Путь |
|----------|------|
| Отчёт валидации | `docs/reports/validation-report.md` |
| RTM | `docs/rtm.md` |

---

## Качественные ворота

### ALL_GATES_PASSED

| Ворота | Статус |
|--------|--------|
| PRD_READY | ✓ |
| RESEARCH_DONE | ✓ |
| PLAN_APPROVED | ✓ |
| IMPLEMENT_OK | ✓ |
| REVIEW_OK | ✓ |
| QA_PASSED | ✓ |

Дополнительно:
- [ ] Все артефакты существуют
- [ ] RTM актуальна

---

## Примеры использования

```bash
# После /test
/validate
```

---

## Следующий шаг

После прохождения ворот `ALL_GATES_PASSED`:

```bash
/deploy
```
